# KnowledgeVault Backend

This is a backend REST API for a notes management application built with Node.js, Express.js, and MongoDB.

The project started as a way to learn backend development, but I gradually added features that are commonly used in real-world applications like JWT authentication, soft delete, pagination, favorites, and Swagger API documentation.

## Features

- User registration and login
- JWT authentication
- Create, update and delete notes
- Search notes by title
- Filter notes by category
- Pagination and sorting
- Mark notes as favorite
- Soft delete and restore
- Permanent delete
- Swagger API documentation

## Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs
- Swagger
- Render
